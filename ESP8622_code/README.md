# ESP8266 Configuration

Replace the placeholder values in `src.ino` with your actual credentials:

- `YOUR_WIFI_SSID` - Your WiFi network name
- `YOUR_WIFI_PASSWORD` - Your WiFi password
- `YOUR_AIO_USERNAME` - Your Adafruit IO username
- `YOUR_AIO_KEY` - Your Adafruit IO key

**Important:** Never commit your actual credentials to GitHub!
